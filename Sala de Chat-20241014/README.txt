1) Primero se compila el archivo ServidorChat (javac ServidorChat.java) y luego "java ServidorChat"
2) En otra Consola o terminal se compila ClienteChat(javac ClienteChat.java) y luego "java ClienteChat"
3) Se ingresa la cantidad de cleintes on las que se desea trabajar
4) Asigna un grupo a cada cliente creado ingresando el nombre del grupo(si se quiere que mas de un cliente esten en el mismo grupo solo hay que repetir el nombre del grupo).
5) Para enviar un mensaje general solo se escribe el mensaje y se envia.
6) Para enviar un mensaje al grupo se envia el mensaje con el simbolo "#" antes del mensaje.Ej: "#hola grupo".
7) Para enviar un privado hay que usar el simbolo "@" seguido del id del cliente que deseamos contactar y finalmente ":".Ej: "@2: hola liente 2"
